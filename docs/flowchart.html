🔄 Common Workflows

Workflow 1: Complete Stream Assessment

1. 📐 Technical Manual → Review appropriate methods for your stream type
2. 🔧 User Manual → Set up project and tools
3. 🔧 User Manual → Collect field data following protocols
4. 🔧 User Manual → Process data in ArcGIS Pro
5. 🔧 User Manual → Run analysis workflows
6. 📐 Technical Manual → Validate results against expected ranges
7. 🔧 User Manual → Generate reports
8. 📐 Technical Manual → Write methods section for deliverable

Workflow 2: Learning a New Method

1. 📐 Technical Manual → Read theory and equations
2. 🔧 User Manual → Download example dataset
3. 🔧 User Manual → Follow tutorial step-by-step
4. 📐 Technical Manual → Understand why results look the way they do
5. 🔧 User Manual → Apply to your own data
6. 📐 Technical Manual → Validate your results

Workflow 3: Troubleshooting an Issue

1. 🔧 User Manual → Check troubleshooting section
2. 🔧 User Manual → Verify workflow steps
3. 📐 Technical Manual → Confirm method is appropriate
4. 📐 Technical Manual → Check if results are within expected ranges
5. 🔧 User Manual → Review example to compare

💡 Pro Tips

For Best Results:

  1. Start with the right manual for your immediate need
  2. Cross-reference when you need deeper understanding
  3. Bookmark frequently used sections in both
  4. Keep both open when doing analysis
  5. Verify versions match (e.g., both v1.2.0)

Red Flags:

  • ❌ Using only one manual for everything
  • ❌ Skipping validation steps
  • ❌ Not understanding the “why” behind methods
  • ❌ Following workflows without checking appropriateness

Green Flags:

  • ✅ Consulting Technical Manual before starting
  • ✅ Following User Manual workflows step-by-step
  • ✅ Validating results with Technical Manual
  • ✅ Understanding both theory and practice
